Hey there 👋,

I'm **Tobias Steckenborn** 👨‍🚀, an **SAP consultant specializing** in **business process improvement** with the **SAP Business Technology Platform**.

I **believe** in a future where technology is used to simplify and automate everyday business processes.

My **mission** is to give my customers the time to focus on what's really important.


<details>
  <summary><b>:telescope: 2021 Goal</b></summary>
  I want to empower others to become SAP Business Technology Platform Developers.<br><br>To achieve this goal I'm currently working on creating courses and tutorials <a href="https://www.udemy.com/user/tobias-steckenborn/">on Udemy</a> following a journey outlined in <a href="https://blogs.sap.com/2020/12/10/how-to-become-a-sap-cloud-platform-developer-in-2021/">this blog post</a>, which describes one possible route to learn most important areas for backend development on SAP BTP. I'm hoping to finish a first MVP of each area this year so you can actually follow the journey end-to-end and get ready for SAP BTP without much hassle.
</details>
